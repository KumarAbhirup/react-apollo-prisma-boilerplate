# 💃 Documentation

## Install and Access the environment
- **Step 1:** Install the ZIP file of this GitHub repository.
- **Step 2:** Open `/frontend` and `/backend` directory in separate Terminal tabs.
- **Step 3:** Run `npm i` in both directories to install necessary modules.
- **Step 4:** [Setup your Backend](#SetupBackend).
- **Step 5:** Run `npm run dev` in both directories.
- **Step 6:** Visit [localhost:3000](http://localhost:3000) and take a breathe. 😀
